# Richpear Home Assistant Add-ons Repository

Tento repozitar je pripraveny pro pridani do Home Assistant Add-on Store jako custom repository.

## Obsah

- `richpear_secure_tunnel` - add-on pro registraci a bezpecny outbound tunnel do Richpear infrastruktury.

## Jak pridat do Home Assistant

1. Home Assistant -> Settings -> Add-ons -> Add-on Store.
2. Vpravo nahoře kliknout na tri tecky -> Repositories.
3. Pridat URL tohoto repozitare.
4. Nainstalovat `Richpear Secure Tunnel`.
